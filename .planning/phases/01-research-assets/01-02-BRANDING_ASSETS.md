# Branding Assets & Social Preview Design Spec

## Aqua-AI Logo Variants
The logo represents water intelligence, combining a blue water drop with an AI node/triangle (A).

- **Source File**: `assets/logo.svg`
- **Primary Color Palette**:
  - Deep Blue: `#0077b6`
  - Sky Blue: `#90e0ef`
  - Clean White: `#ffffff`
- **Typography**: Clean Sans-Serif (e.g., Inter, Montserrat).

## Dashboard Screenshot Specification
To create a high-tier repository image, capture screenshots of these key views:

### 1. Main Dashboard Overview
- **View**: General stats, real-time data overview.
- **States**: Populate with realistic, high-quality data. Use mock data that looks "active".
- **Visuals**: Dark mode preferred for professional AI look. Highlight data visualizations (charts, maps).

### 2. AI Intelligence Engine
- **View**: A screen showing the 'Intelligence' or 'Insights' tab.
- **States**: Display an AI-generated warning or notification about water quality or usage anomalies.
- **Visuals**: Highlight the specific "AI recommendation" card or element.

### 3. Data Visualization
- **View**: A high-contrast chart view showing historical data comparisons.
- **States**: At least 3 data series visible to show complexity and depth.
- **Visuals**: Smooth, professional gradients on charts.

## Social Preview Image Design
This image appears when the GitHub link is shared on platforms like LinkedIn or X (formerly Twitter).

- **Dimensions**: 1280x640px (GitHub standard).
- **Layout**: 
  - **Left Side**: Large Aqua-AI Logo and Name.
  - **Center/Right Side**: Mockup of the Dashboard within a clean browser or tablet frame.
  - **Bottom Left**: Tagline: "AI-Powered Water Intelligence for Global Sustainability".
  - **Overlay**: GitHub Star button or "Star on GitHub" text to prompt action.
- **Background**: Deep blue gradient background (#001d3d to #003566) with a subtle "data grid" or "water ripple" pattern.
- **Tone**: High-tech, professional, trustworthy, sustainable.
