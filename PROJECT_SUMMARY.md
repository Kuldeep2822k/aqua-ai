## 🔧 Quick Start Commands

```bash
# Frontend Demo (runs immediately)
cd frontend
npm install
npm start
# Visit: http://localhost:3000

# Backend API
cd backend  
npm install
npm run dev
# API: http://localhost:5000

# AI Model Training
cd ai-models
python train_model.py

# Data Pipeline
cd data-pipeline  
python fetch_data.py

# Full Docker Setup
docker-compose up -d
```
